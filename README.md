# Sales Data Analysis

## 📌 Objective
The objective of this project is to analyze sales data and extract meaningful business insights.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed
- Sales by Region
- Sales by Category
- Monthly Sales Trends
- Top Products Analysis

---

## 📈 Key Insights
- West region has the highest sales
- Technology category contributes the most revenue
- Sales peak during November–December
- A small number of products generate a large portion of revenue

---

## 📌 Conclusion
This project demonstrates data cleaning, analysis, visualization, and extracting actionable insights from raw data.
